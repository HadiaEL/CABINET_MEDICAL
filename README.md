# CABINET_MEDICAL
En tant que patient:  je peux consulter la liste des médecins disponibles.  Chaque médecin est associé à une spécialité. je peux sélectionner un médecin ainsi que de visualiser les créneaux de ses rendez-vous disponibles et réserver un créneau horaire disponible. Chaque créneau ne peut être réservé que par un seul patient.
